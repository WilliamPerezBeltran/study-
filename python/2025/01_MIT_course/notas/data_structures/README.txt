* Categorias principales de estructuras de datos segun su organización:

- lineales:
		- Se recorre de inicio a fin en un solo camino.
		- arrays, listas, pilas, colas, listas enlazadas
- No lineales:
		- Los datos tienen multiples relaciones 
		- arboles, grafos 
- tabulares/ asociativas:
		- Basadas en clave-valor o en tablas 
		- hash tables (diccionarios, mapas), matrices 
- Estructuras jerarquicas:
		- Organizacion por niveles   
		- arbol nivario, arbol B, Heap
- Estructuras de datos dinamicas vs estaticas:
		- Estaticas tamaño fijo ( arrays )
		- Dinamicas : tamañno flexible (listas enlazadas diccionarios)
